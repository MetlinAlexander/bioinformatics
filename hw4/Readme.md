## Краткая инструкция по запуску
1. Скачать `1A3N.pdb` с https://www.rcsb.org/structure/1A3N в формате Legacy PDB (.pdb).
2. Открыть Chimera → File → Open → выбрать `1A3N.pdb`.
3. Следовать шагам из `steps.md` для получения изображений.
